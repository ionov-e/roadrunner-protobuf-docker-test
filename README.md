# Small testing repository: Roadrunner + Protobuf (using Docker)

![Roadrunner Schema](documentation/roadrunner_schema.png)

## Instruction

1.  Copy `.env` file
    ```bash
    cp .env.dist .env
    ```
2. Edit `.env`
3. Run
    ```bash
    docker compose up -d
    ```
4. Open in browser http://localhost:5000/
